Starter boilerplate for an universal WebApp using React, Redux, Router, LoopBack and MongoDB.
Download
Documentation

Client

For the client side we are using React, Redux and Router to this starter boilerplate for an universal WebApp.

Server

For the server side we are using LoopBack. LoopBack is a highly-extensible, open-source Node.js framework.

DataBase

For database we are using MongoDB, as it a document-based data model.

Quick Start 



git clone git@github.com:opprDev/R3LM.git
cd R3LM/
npm install

To install and use this repository npm is needed. Please install both Node.js and npm first. Read more.
